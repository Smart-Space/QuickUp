# QuickUp 窗口排布新行为

QuickUp v4.5起，被排布的窗口经过放大再还原后，会回到排布之前的位置和尺寸，与Windows Snap功能一致。

> QuickUp 4.5起，不再更新源码。这有很多考虑，但是此前代码的QuickUp，完整度已经十分高，后续的功能优化不再跟进。

---

# QuickUp window layout new behavior

Starting from QuickUp v4.5, after the arranged window is enlarged and restored, it will return to its previous position and size, which is consistent with the Windows Snap function.

> Starting from QuickUp 4.5, the source code will no longer be updated. There are many considerations for this, but the integrity of the previous QuickUp code is already very high, and subsequent functional optimizations will no longer be followed up.
